# 演習3

・ ユーザー入力値をブラウザ表示する場合、PHPの「htmlspecialchars」関数でエスケープする。
・エスケープ処理をしていないと、「クロスサイトスクリプティング」と呼ばれる脆弱性の原因となる。
・PHPの「htmlspecialchars」関数は引数の指定が「冗長」なので、「ユーザー定義関数」を介して短く呼び出す。

**語群：**
html.escape / htmlspecialchars / htmlEscape / HtmlEncode /
ブルートフォース攻撃 / スニッフィング / クロスサイトスクリプティング /
冗長 / 簡潔 / 抽象的 / コールバック関数 / 無名関数 / ユーザー定義関数 /
