# 演習3

## クラス／オブジェクト／インスタンスの理解

| 問題文                                                           | 真偽値 |
| ---------------------------------------------------------------- | ------ |
| 同じクラスから、複数のオブジェクトが作られることがある           | true   |
| クラス自体に、いろいろな場所から別々の値を書き込んで使うのが基本 | false  |
| オブジェクトは自分専用のデータ領域にデータを保持できる           | true   |
| クラスは「データ型」、インスタンスは「実際の値」に例えられる     | true   |
